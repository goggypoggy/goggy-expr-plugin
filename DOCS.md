- **Возвращает:** `none`

---

### `deactivate()`
**Деактивация расширения**

- **Аргументы:** `none`

---

### `ExpandWord(keyword)`
**Проверяет и дополняет ключевое слово**

- **Аргументы:**
  - `keyword`: `string` - ключевое слово, которое нужно раскрыть
- **Возвращает:**
  - `true` - если ключевое слово было раскрыто
  - `false` - в противном случае
- **Описание:** Проверяет, было ли набрано ключевое слово в редакторе и был ли после него пробел. Если да, раскрывает ключевое слово в полное выражение.
- **Пример вызова:**
```
bool if_was_expanded = ExpandWord("if");
// если курсор стоит на слове "if" и был написан пробел,
// то в активном редакторе будут вставлены скобки
// if_was_expanded будет true
```

---

### `ExpandWords(word_list)`
**Пытается дополнить каждое ключевое слово из списка**

- **Аргументы:**
  - `word_list`: `string Array` - массив ключевых слов для дополнения
- **Возвращает:**
  - `true` - если какое-либо ключевое слово из списка было раскрыто
  - `false` - если ни одно ключевое слово не было раскрыто
- **Описание:** Перебирает список ключевых слов и пытается дополнить каждое. Останавливается после первого успешного дополнения.
- **Пример вызова:**
```
const keywords = ["if", "for", "while"];
bool smth_was_expanded = ExpandWord(keywords);
// если слово, на котором стоит курсор, есть в массиве keywords и был написан пробел,
// то в активном редакторе будут вставлены скобки
// smth_was_expanded будет true
```

## Слушатели событий

### Обработчик `onDidChangeTextDocument`
**Слушатель событий изменения текста документа**

- **Аргументы:**
  - `event`: `vscode.TextDocumentChangeEvent` - событие изменения документа
- **Возвращает:** `None`
- **Описание:** Точка расширения, где проверяются раскрываемые ключевые слова и выполняется их дополнение при возможности.

---

## Команды расширения

### `goggy-expr-plugin.enable`
**Включение автодополнения выражений**

- **Название:** "Enable Expression Auto-completion"
- **Доступ:** Через палитру команд (Command Palette)
- **Функция:** Включает расширение, если оно было отключено, выводит сообщение "Expression auto-completion enabled"

---

### `goggy-expr-plugin.disable`
**Отключение автодополнения выражений**

- **Название:** "Disable Expression Auto-completion"
- **Доступ:** Через палитру команд (Command Palette)
- **Функция:** Отключает расширение, если оно было включено, выводит сообщение "Expression auto-completion disabled"

---

### `goggy-expr-plugin.toggle`
**Переключение автодополнения выражений Вкл/Выкл**

- **Название:** "Toggle Expression Auto-completion On/Off"
- **Доступ:** 
  - Через палитру команд (Command Palette)
  - Горячая клавиша: `Ctrl+Alt+E`
- **Функция:** Вызывает соответствующую команду в зависимости от текущего состояния расширения
