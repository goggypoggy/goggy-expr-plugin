## Описание

Данный плагин автоматически генерирует блоки if, for и while, когда в редакторе пишутся if, for и while. Табуляция сохраняется. После генерации блока курсор переносится в скобки условия. Блоки написаны в [Google Code Style](https://google.github.io/styleguide/cppguide.html). Плагин можно включать и выключать по надобности.

## Команды

+ `Enable Expression Auto-completion`

  Включает плагин.

+ `Disable Expression Auto-completion`
  
  Выключает плагин.

+ `Toggle Expression Auto-completion On/Off`

  Включает/Выключает плагин.\
  Горячие клавиши:
  - Windows и Linux: `Сtrl+Alt+E`
  - Mac: `Cmd+Alt+E`  

> Выключение плагина не деактивирует его.

> Все эти команды можно использовать через палитру команд.
